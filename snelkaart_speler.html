Greyhawk Campaign Manager

SNELKAART SPELER — bewaar dit naast je scherm

Eerste keer

1. AccountRegistreer op de app-link van de DM. DM Code = leeg laten.
2. KarakterKlik + Nieuw Karakter. Kies ras → klasse → gezindheid.
3. StatsKlik op elk getal met stippellijn → typ → Enter. Alles slaat automatisch op.
4. Foto📷 icoontje op je character sheet. Max 2MB.

Karakter bewerken

Waarde wijzigenKlik op stippellijn-getal → typ → Enter
Wapen toevoegen"+ wapen toevoegen" → zoek → klik → Toevoegen
Item/skill/spellIdem via de + knoppen
PDF exporteren📄 PDF knop op character sheet
Alle export📤 Export → kies JSON/MD/PDF

Sessie & gevecht

Sessie bekijkenTab Sessies → klik op sessie
Log invullen"+ Log" → ontmoetingen, NPCs, loot, notities
InitiativeBij gevecht: vul je d6 + DEX mod in het veld
🎲 RollenIn sessie: "🎲 Gooi dobbelsteen". 🔒 = privé (alleen jij + DM)
💬 ChatIn sessie: typ bericht. Kies: iedereen / fluister DM / fluister speler

Aanval doen (AD&D)

1. Gooi D20
2. Tel bonussen op (STR, magie)
3. D20 + bonus ≥ THAC0 − vijand AC → RAAK!
4. Gooi schade-dobbelsteen (staat bij wapen)
5. Tel schadebonussen op
