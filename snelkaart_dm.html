Sessie aanmaken

1.Tab Sessies → + Nieuwe sessie
2.Naam + datum + locatie invullen
3.Karakters aanvinken die meedoen
4.Status: Gepland → Actief → Afgesloten

Gevecht starten

1.Open sessie → ⚔️ Start gevecht
2.Spelers vullen zelf initiative in (d6+DEX)
3.🎲 Gooi init voor DM-karakters
4.↕ Sorteer (laagst = eerst)
5.Per karakter: 📋 Actie → wat doet het + resultaat
6.Ronde X → voor volgende ronde
7.✓ Einde gevecht

Tijdens gevecht

HP aanpassenTyp direct in HP-veld
StatusDropdown: Actief/Bewusteloos/Dood/Gevlucht
+ VijandNaam + aantal + HP → aparte entries
🎮 OvernemenKlik 🎮 → "DM" of andere spelernaam

XP toekennen

1.Open sessie → ⭐ XP per karakter
2.Bedrag + reden → automatisch in karakter + logboek

Wachtwoord resetten

1.🔑 Reset rechtsboven
2.Kies speler → tijdelijk wachtwoord
3.Speler wordt verplicht nieuw wachtwoord te kiezen
