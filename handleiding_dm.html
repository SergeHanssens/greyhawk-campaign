Greyhawk Campaign Manager

Dungeon Master Handleiding — Vertrouwelijk · Niet delen met spelers
DM Code: GREYHAWK_DM_2025 · Deel dit NIET met spelers · App: greyhawk-frostbraid.netlify.app

1. DM Account aanmaken

1
Ga naar de app-link
2
Klik "Nieuw account aanmaken"

Kies gebruikersnaam (bv. bart_dm) en wachtwoord (min. 6 tekens).

3
Vul de DM Code in

Typ GREYHAWK_DM_2025 in het DM Code veld. Vervolgens verschijnt een extra sectie voor je veiligheidsvraag.

4
Stel een veiligheidsvraag in

Kies een vraag die alleen jij weet, bv. "Wat is de naam van mijn eerste karakter?" + antwoord. Dit gebruik je als je ooit je wachtwoord vergeet.

5
Log in → je ziet "Dungeon Master" badge + extra tabblad "DM Dashboard"

2. DM Dashboard

Het tabblad DM Dashboard toont alle karakters van alle spelers in één overzicht.

Wat je zietBetekenis
HP balk groenSpeler is gezond (>50% HP)
HP balk geelSpeler gewond (25-50% HP)
HP balk roodSpeler in gevaar (<25% HP)
Uitgevaagde kaartKarakter staat op inactief
🔒 preview tekstJouw privé DM-notitie voor dit karakter

Klik op een karakterkaart om het volledig te openen en te bewerken.

3. Karakters bewerken als DM

Als DM kan je alle karakters van alle spelers bekijken en aanpassen:

✓ HP aanpassen (schade na gevecht, genezing)
✓ XP toekennen na sessie
✓ Items/wapens/vaardigheden/spreuken toevoegen of verwijderen
✓ Alle statistieken aanpassen (level-up)
✓ Privé DM notities schrijven
✓ Sessie-aantekeningen bijhouden
✓ Karakters inactief zetten of verwijderen

HP aanpassen na gevecht

1
Open het karakter via DM Dashboard
2
Klik op het HP getal

Het veld wordt bewerkbaar met een stippellijn. Typ het nieuwe getal.

3
Druk Enter of klik ergens anders

Automatisch opgeslagen. De speler ziet dit bij de volgende login.

4. DM Privé Notities & Sessie-aantekeningen

Privé Notities: onderaan elk karakterblad in een paars kader. Alleen jij ziet dit. Gebruik voor: geheimen over het karakter, geplande verhaalontwikkelingen, zwaktes...
Sessie-aantekeningen: tweede paars kader. Alleen jij ziet dit. Gebruik voor: bijhouden wat de speler deed deze sessie, op basis waarvan je XP gaat toekennen, interessante momenten voor verhaalontwikkeling.

5. Logboek

Klik 📋 Logboek op een karakter voor een volledig overzicht van alle wijzigingen (wie, wat, wanneer, oude → nieuwe waarde).

Spelers zien automatisch jouw wijzigingen bij de volgende keer dat ze het karakter openen. Ze hoeven niet zelf het logboek te checken.

6. Wachtwoord resetten van speler

1
Klik "🔑 Reset" rechtsboven in de app
2
Kies de speler en stel een tijdelijk wachtwoord in

Bv. Greyhawk2025 — geef dit mondeling mee aan de speler.

3
De speler logt in met dit tijdelijk wachtwoord

Ze worden automatisch verplicht een nieuw persoonlijk wachtwoord te kiezen.

7. Encyclopedie beheren

Als DM zie je in de Encyclopedie tab een ✏ bewerk-icoon naast elk item en een + Toevoegen formulier onderaan.

ActieHoe
Item bewerkenKlik ✏ → pas velden aan → Opslaan
Item verwijderenKlik ✏ → Verwijderen knop
Nieuw item toevoegenKies categorie → vul formulier in onderaan → Toevoegen
Homebrew monsterTab Monsters → + formulier onderaan
📥 CSV importerenKlik 📥 CSV importeren rechtsboven het + formulier. Upload een CSV-bestand of plak inhoud direct. Eerste rij = kolomnamen.

Alles wat je toevoegt is meteen beschikbaar voor spelers als ze wapens/items zoeken.

CSV importeren — voorbeeld

Per categorie zijn er verwachte kolommen (zie tooltip in het importvenster). Voorbeeld voor monsters:

name,ac,hp_dice,thac0,damage,move,alignment,description Goblin Chief,4,3+1,18,1d8+2,9,"Lawful Evil",Leider van een goblin-clan Wolf Pack Alpha,6,3+3,16,1d6+2,18,"Neutral",Grootste wolf in een roedel

Tekstvelden met komma's: tussen aanhalingstekens ("…"). Onbekende kolommen worden genegeerd. Kolom name is altijd verplicht.

7b. Scan-functie voor karakterbladen 📷

Spelers (en jij) kunnen een foto van een handgeschreven karakterblad inscannen. Claude (Anthropic API) leest de gegevens automatisch uit en stelt een nieuw karakter voor.

Vereist: Deze functie werkt enkel als de app op Netlify draait (niet op gewone GitHub Pages) én als ANTHROPIC_API_KEY als environment variable is ingesteld.
Setup (eenmalig): zie README onder "Optioneel: scan-functie via Netlify". Geschatte kost: gratis Netlify tier + ~€0,01 per scan via Anthropic API.

8. Karakter inactief / verwijderen

Inactief: Klik ⏸ Inactief op het karakterblad. Het karakter is nog bewaard maar verdwijnt iets uit het zicht. Kan altijd opnieuw geactiveerd worden met ✓ Activeren.

Verwijderen: Klik 🗑 Verwijderen → bevestigingsvenster → definitief. Alle wapens, items, het logboek — alles wordt gewist. Niet ongedaan te maken!

9. Supabase — directe database toegang

Voor bulk-wijzigingen of technische problemen: ga naar supabase.com → jouw project → Table Editor.

Nuttige tabellen:
• players — alle gebruikers (zet is_dm = true om iemand DM te maken)
• characters — alle karakteren
• character_log — volledig wijzigingslogboek
• weapons/spells/items/monsters — de encyclopedie

10. GitHub & Updates

De app staat op GitHub. Updates worden automatisch actief zodra Serge het nieuwe HTML-bestand uploadt. Spelers hoeven niets te doen.

Als de app offline gaat of iets kapot gaat: contacteer Serge — hij beheert de technische kant.
