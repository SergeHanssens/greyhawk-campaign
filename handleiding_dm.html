Greyhawk Campaign Manager

Dungeon Master Handleiding — versie 1.7 — Vertrouwelijk

1. DM Account aanmaken

1
Registreer met de DM Code

Vul bij registratie de DM Code in → extra vak verschijnt voor veiligheidsvraag.

2
Stel veiligheidsvraag in

Bv. "Naam van mijn eerste karakter?" — gebruik dit als je je wachtwoord vergeet.

3
Na login: "Dungeon Master" badge + extra tabs

2. DM Dashboard

Tab DM Dashboard toont alle karakters van alle spelers. HP-balken (groen/geel/rood), inactieve karakters uitgevaagd, en een preview van je privé DM-notities per karakter.

3. Karakters bewerken

Als DM kan je alle karakters bekijken en bewerken:

HP/XP/stats aanpassen · Wapens/items/spreuken toevoegen/verwijderen · Profielfoto uploaden · Inactief zetten of verwijderen · Privé DM-notities schrijven · Sessie-aantekeningen bijhouden

4. DM Privé Notities

Twee velden per karakter, onzichtbaar voor spelers:
Privé Notities — geheimen, zwaktes, verhaalplannen
Sessie-aantekeningen — voor XP-berekening en verhaalontwikkeling

5. Sessies aanmaken & beheren 📅

1
Tab Sessies → + Nieuwe sessie

Naam, datum, locatie, samenvatting (publiek), DM-notities (privé).

2
Selecteer deelnemers

Vink aan welke karakters meedoen. Inactieve karakters staan er ook bij (voor flashbacks).

3
Status

Gepland → Actief → Afgesloten. Wijzig wanneer je wilt.

6. Gevechten beheren ⚔️

Open een sessie → klik ⚔️ Start gevecht. Het gevechts-panel verschijnt.

Initiative

Spelers vullen zelf hun initiative-worp in (d6 + DEX modifier) via het invulveld naast hun karakter.
DM klikt "🎲 Gooi init voor DM-karakters" → rolt alleen voor vijanden en NPCs die jij bestuurt.
Sorteer → laagste initiative = eerste beurt (AD&D 1e regel).

Rondes & acties

1
Per karakter: 📋 Actie

Klik om vast te leggen wat het karakter doet: aanval, spreuk, verdediging, vlucht. Plus het resultaat (raak/mis, schade). Wordt bewaard per ronde.

2
Ronde X →

Klik om naar de volgende ronde te gaan. Actielog van de vorige ronde blijft zichtbaar.

3
✓ Einde gevecht

Sluit het gevecht af.

Tijdens het gevecht

ActieHoe
HP aanpassenTyp direct in het HP-veld naast elk karakter
Status wijzigenDropdown: Actief / Bewusteloos / Dood / Gevlucht
+ Vijand toevoegenKlik "+ Vijand" → naam + aantal + HP. Worden als aparte entries toegevoegd
Wie bestuurt wieKlik 🎮 → verander naar "DM", andere spelernaam, of "owner" (terug naar eigen speler)

Karakter overnemen / doorgeven

Bij afwezigheid van een speler: klik 🎮 naast het karakter → typ de naam van de speler die het overneemt, of "DM" als jij het speelt. Meerdere karakters per speler is geen probleem.

7. XP toekennen ⭐

1
Open sessie → ⭐ XP naast een karakter
2
Bedrag + reden invullen

Reden komt in het persoonlijk logboek van de speler ("Sessie-XP: +500 (hydra verslagen)").

3
Karakter XP wordt automatisch bijgewerkt

8. Wachtwoord resetten

Klik 🔑 Reset rechtsboven → kies speler → stel tijdelijk wachtwoord in. Speler wordt bij login verplicht een nieuw te kiezen.

9. Encyclopedie beheren

ActieHoe
Bewerken✏ knop naast elk item → pas velden aan → Opslaan
Verwijderen✏ → Verwijderen knop
ToevoegenFormulier onderaan Encyclopedie pagina
AfbeeldingIn edit/add: 🖼 veld → URL plakken of bestand uploaden (max 2MB)
Bulk import📥 Importeren → CSV, JSON of Markdown. Kolom name verplicht.
Tip — snel 100 entries toevoegen: vraag aan ChatGPT/Claude: "Geef me 50 AD&D 1e monsters in JSON-array met velden: name, ac (int), hp_dice, thac0 (int), damage, move (int), alignment, description". Plak in 📥 Importeren → JSON → klaar.

10. Kaart & Mythic Table

Tab Kaart: link naar Mythic Table (speelveld met tokens, fog of war) + Greyhawk referentiekaarten. DM kan:

  • Mythic Table link aanpassen (voor als je een andere sessie-URL hebt)
  • Eigen kaart uploaden (JPG/PNG, max 5MB) of externe URL plakken

11. Export & import

📤 Export op karakter, sessie of encyclopedie: JSON, Markdown, PDF of CSV.

📥 Import op encyclopedie: CSV, JSON of Markdown-tabel.

12. Tools

Tab Tools: 🎲 Dobbelstenen roller, 🎯 THAC0 calculator, ⭐ XP-per-monster tabel.
