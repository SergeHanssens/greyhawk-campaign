Greyhawk Campaign Manager

Dungeon Master Handleiding — versie 3.0 — Vertrouwelijk

1. Accounts beheren

DM-account (eerste keer)

Loginscherm → onderaan klein knopje "DM: eerste account aanmaken" → registreer met DM Code + veiligheidsvraag.

Speler-accounts aanmaken

Tab Mijn Karakters → 👤 Nieuwe speler → gebruikersnaam + tijdelijk wachtwoord. Speler moet bij eerste login een eigen wachtwoord kiezen.

Spelers kunnen zelf GEEN account aanmaken. Alleen de DM.

2. Karakters aanmaken

Tab Mijn Karakters → + Nieuw Karakter. Alleen de DM kan karakters aanmaken.

TypeGebruik
Speler-karakterKies eigenaar (speler) uit dropdown. Ras/klasse volgens AD&D regels.
NPCMonster, vijand, compaan, stadswacht, etc. Eigenaar = DM. Vrije ras/klasse keuze.

De Karakters-pagina splitst automatisch in ⚔ Speler Karakters en 🏰 NPC's.

3. DM Dashboard

Tab DM Dashboard: overzicht alle karakters van alle spelers met HP-balken.

Per karakter: Privé Notities (geheimen, plannen) + Sessie-aantekeningen (XP-tracking). Onzichtbaar voor spelers.

4. Sessies

Aanmaken

Tab Sessies → + Nieuwe sessie → naam, datum, locatie, deelnemers selecteren. Status = Gepland.

Status-workflow

StatusBetekenisSpeler ziet?
GeplandVoorbereiding — DM kan alles instellen❌ Nee
ActiefSpel loopt — spelers nemen deel✅ Ja
AfgeslotenAlleen-lezen — archief✅ Ja (lezen)

▶ Activeren: als er al een actieve sessie is → waarschuwing om die eerst af te sluiten.

Afgesloten: niets meer bewerkbaar tenzij DM op "✏ Bewerken" klikt.

5. Acties binnen een sessie

Klik + Nieuwe actie in een actieve sessie. Types: Gevecht, Verkenning, Sociaal, Rondtrekken, Rust, Anders.

Initiative toggle

Checkbox "Initiative volgorde" — standaard aan. Vink uit voor acties zonder initiative (herbergbezoek, rust).

Meerdere acties tegelijk

Party split? Maak twee aparte acties aan met verschillende deelnemers.

6. Gevechtsrondes

Ronde-flow

1. Ronde X → klik → alle initiative velden worden gereset
2. Spelers rollen eigen initiative (🎲 of typen) → vergrendeld na invullen
3. 🎲 DM init → rolt d20 voor ALLE DM-karakters in één klik
4. Statusbalk: "⏳ Wachten..." → "✓ Ronde loopt!"
5. Per karakter: speler declareert actie in invulveld
6. DM vult resultaat aan via DM-notitie veld
7. Volgende ronde → herhaal

Uitgesteld / Extra aanval

Per karakter checkboxes:

☐ UitgesteldKarakter stelt actie uit → komt einde ronde aan bod
☐ +AanvalExtra aanval aan einde ronde (bv. Fighter 3/2)

Gemarkeerde karakters verschijnen in geel blok "EINDE RONDE".

Status per karakter (DM dropdown)

ActiefNormaal — blauw (speler) of rood (NPC)
KOBewusteloos — oranje rand + gele achtergrond
DoodDood — donkerrood + 50% transparant + 💀
WegGevlucht — grijs
VerborgenOnzichtbaar voor spelers — paars. Voor hinderlagen.

Spelers zien status als label (niet aanpasbaar).

Ronde-navigatie

Knoppen [1] [2] [3]... verschijnen — klik om logs van eerdere rondes te bekijken.

Afgesloten acties

Klik driehoekje → lijst afgesloten acties → 📋 Bekijk logs → ronde-per-ronde overzicht.

8. XP toekennen

In sessie → ⭐ XP per karakter → bedrag + reden. Updates karakter.xp + komt in logboek.

9. Dobbelsteen & Chat

🎲 Rollen: in sessie via knop. Iedereen ziet. 🔒 Privé = jij ziet ALLES (ook privé worpen van spelers).

💬 Chat: iedereen/fluister/groep. DM ziet ALLE berichten — ook fluisterberichten tussen spelers.

10. Encyclopedie

✏ bewerken, + toevoegen, 🖼 afbeelding uploaden, 📥 bulk import (CSV/JSON/MD).

Tip: vraag AI om "50 AD&D monsters als JSON-array" → plak in 📥 JSON import.

11. Kaart & VTT

Tab Kaart: Mythic Table link (aanpasbaar), DM kaart upload, Greyhawk referentiekaarten.

12. Export & Import

📤 Sessie-export bevat ALLES: acties, ronde-logs, dice rolls, chat, deelnemers, XP.

📥 Import encyclopedie: CSV, JSON of Markdown.

13. Wachtwoord resetten

🔑 Reset rechtsboven → kies speler → tijdelijk wachtwoord.

14. Database reset

Voor frisse start: draai sql/99_reset_game_data.sql in Supabase. Verwijdert alle spelers/karakters/sessies. Encyclopedie blijft.
