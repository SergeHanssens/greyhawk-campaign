DM Testgids

v3.0 — Stap-voor-stap alles testen (45 min) — draai eerst sql/99_reset_game_data.sql voor een frisse start

Stap 1 — DM Account aanmaken (2 min)

Test 1.1

  1. Open de app → loginscherm
  2. Klik het kleine knopje onderaan: "DM: eerste account aanmaken"
  3. Gebruikersnaam + wachtwoord + DM Code invullen + veiligheidsvraag
  4. ✓ Verwacht: "Dungeon Master" badge + DM Dashboard tab

Stap 2 — Speler-account aanmaken (2 min)

Test 2.1

  1. Tab Mijn Karakters → 👤 Nieuwe speler
  2. Naam: "testspeler", Wachtwoord: "Test123"
  3. ✓ Verwacht: "✓ Speler testspeler aangemaakt"

Test 2.2 — Test als speler

  1. Log uit → log in als "testspeler" met "Test123"
  2. ✓ Verwacht: verplicht wachtwoord wijzigen
  3. Kies nieuw wachtwoord → ✓ ingelogd als speler
  4. Controleer: geen DM Dashboard, geen + Nieuw Karakter, geen 👤 Nieuwe speler
  5. Log uit → log weer in als DM

Stap 3 — Karakters aanmaken (3 min)

Test 3.1 — Speler-karakter

  1. + Nieuw Karakter → Type: Speler-karakter
  2. Eigenaar: "testspeler" → Naam: "Thorin" → Dwarf Fighter
  3. ✓ Verwacht: verschijnt bij ⚔ SPELER KARAKTERS

Test 3.2 — NPC

  1. + Nieuw Karakter → Type: NPC
  2. Naam: "Goblin Sluipschutter" → Ras: Monster → Klasse: Enemy
  3. ✓ Verwacht: verschijnt bij 🏰 NPC's met rode banner

Stap 4 — Sessie + Actie (5 min)

Test 4.1 — Sessie aanmaken

  1. Tab Sessies → + Nieuwe sessie
  2. Naam: "Testsessie", Status: Gepland, deelnemers: Thorin + Goblin
  3. ✓ Verwacht: sessie verschijnt in lijst

Test 4.2 — Activeren

  1. Open sessie → klik ▶ Activeren
  2. ✓ Verwacht: status wordt "ACTIEF"

Test 4.3 — Actie starten

  1. + Nieuwe actie → "Hinderlaag" → Type: Gevecht → Initiative: aangevinkt
  2. Deelnemers: alle → Actie starten
  3. ✓ Verwacht: actie verschijnt met "Ronde 1" + init velden

Stap 5 — Initiative + Ronde (5 min)

Test 5.1 — DM initiative rollen

  1. Klik 🎲 DM init
  2. ✓ Verwacht: alle DM-karakters (Goblin) krijgen init-getal

Test 5.2 — Speler initiative

  1. Bij Thorin: klik 🎲 naast INIT veld
  2. ✓ Verwacht: d20 resultaat ingevuld + gesorteerd
  3. ✓ Verwacht: "✓ Alle initiatives ingevuld — ronde 1 loopt!"

Test 5.3 — Actie declareren

  1. Bij Thorin: typ "Aanval op Goblin" → klik ✓
  2. Bij Goblin: DM notitie: "Probeert te vluchten" → DM+
  3. ✓ Verwacht: beide entries verschijnen in ronde-log

Test 5.4 — Nieuwe ronde

  1. Klik Ronde 2 →
  2. ✓ Verwacht: initiative velden leeg + "⏳ Wachten..."

Test 5.5 — Status wijzigen

  1. Zet Goblin op Dood via dropdown
  2. ✓ Verwacht: donkerrode rand + 💀 + half transparant
  3. Zet op Verborgen
  4. ✓ Verwacht: paars + 👁‍🗨 (spelers zien dit niet)
