DM Testgids

Stap-voor-stap alles testen voor je de spelers uitnodigt — verwachte tijd: 30 minuten

Dit document leidt je door elke functie van de Greyhawk Campaign Manager. Test alles in volgorde. Als iets niet werkt, noteer welke stap en welke foutmelding je krijgt.

Stap 1 — Inloggen als DM (2 min)

Test 1.1: Inloggen

  1. Ga naar de app-link
  2. Log in met je DM-account
  3. ✓ Verwacht: "Dungeon Master" badge rechtsboven + extra knoppen (🔑 Reset) + tab "DM Dashboard"

Stap 2 — Karakter aanmaken (5 min)

Test 2.1: Nieuw karakter

  1. Tab Karakters → klik + Nieuw Karakter
  2. Naam: "Testridder", Ras: Human, Klasse: Fighter, Gezindheid: Lawful Good
  3. Klik Karakter Aanmaken
  4. ✓ Verwacht: Character sheet opent met alle velden

Test 2.2: Velden bewerken

  1. Klik op het HP getal (stippellijn) → typ 45 → druk Enter
  2. Doe hetzelfde voor STR (typ 16), DEX (14), THAC0 (17)
  3. ✓ Verwacht: groen "✓ Auto-opslaan" bevestiging, waarden bijgewerkt

Test 2.3: Wapen toevoegen

  1. Klik + wapen toevoegen
  2. Typ "long" in de zoekbalk → "Long Sword" verschijnt
  3. Klik op Long Sword → formulier vult zich in
  4. Klik + Toevoegen aan karakter
  5. ✓ Verwacht: Long Sword verschijnt in de wapenlijst op de sheet

Test 2.4: Profielfoto

  1. Klik het 📷 icoontje linksboven op de character sheet
  2. Kies een afbeelding (max 2MB)
  3. ✓ Verwacht: foto verschijnt als avatar

Stap 3 — DM Dashboard (2 min)

Test 3.1: Overzicht

  1. Klik tab DM Dashboard
  2. ✓ Verwacht: alle karakters zichtbaar met HP-balken

Test 3.2: Privé notities

  1. Open een karakter vanuit DM Dashboard
  2. Scroll naar beneden → paars kader "🔒 DM Privé Notities"
  3. Typ iets → klik ergens anders
  4. ✓ Verwacht: "✓ DM notities opgeslagen" melding. Spelers zien dit NIET.

Stap 4 — Sessie aanmaken + gevecht (10 min)

Test 4.1: Nieuwe sessie

  1. Tab Sessies → klik + Nieuwe sessie
  2. Naam: "Testsessie", Datum: vandaag, Status: Actief
  3. Vink minstens 2 karakters aan
  4. Klik ✓ Opslaan
  5. ✓ Verwacht: sessie verschijnt in de lijst, klik erop om te openen

Test 4.2: Gevecht starten

  1. Open de testsessie → klik ⚔️ Start gevecht
  2. ✓ Verwacht: rood kader verschijnt met "Ronde 1" + initiative-velden per karakter
  3. Typ bij het eerste karakter initiative 3, bij het tweede 5

Test 4.3: Vijand toevoegen

  1. Klik + Vijand → naam: "Goblin", aantal: 3, HP: 7
  2. ✓ Verwacht: "Goblin #1", "Goblin #2", "Goblin #3" verschijnen als rode entries

Test 4.4: DM initiative rollen

  1. Klik 🎲 Gooi init voor DM-karakters
  2. ✓ Verwacht: goblins krijgen een initatiefgetal
  3. Klik ↕ Sorteer → laagste initiative bovenaan

Test 4.5: Actie vastleggen

  1. Klik 📋 Actie bij het eerste karakter
  2. Typ: "Aanval op Goblin #1 met Long Sword" → Resultaat: "Raak! 6 dmg"
  3. ✓ Verwacht: actie verschijnt in "Acties — Ronde 1"

Test 4.6: HP aanpassen + status

  1. Bij Goblin #1: verander HP naar 1
  2. Verander status dropdown naar Dood
  3. ✓ Verwacht: Goblin #1 wordt uitgevaagd met 💀

Test 4.7: Volgende ronde + einde

  1. Klik Ronde 2 →
  2. ✓ Verwacht: "Ronde 2" verschijnt
  3. Klik ✓ Einde gevecht

Stap 5 — XP toekennen (2 min)

Test 5.1: XP

  1. Open de testsessie → klik ⭐ XP bij een karakter
  2. Bedrag: 100, Reden: "3 goblins verslagen"
  3. Klik ✓ Toekennen
  4. ✓ Verwacht: karakter toont "+100 XP deze sessie" + totale XP is bijgewerkt

Stap 6 — Encyclopedie (3 min)

Test 6.1: Zoeken

  1. Tab Encyclopedie → klik Monsters
  2. Zoek "dragon" in de zoekbalk
  3. Klik info naast een resultaat
  4. ✓ Verwacht: popup met naam, stats, beschrijving + afbeelding (als beschikbaar)

Test 6.2: Item bewerken (DM)

  1. Klik ✏ naast een wapen
  2. Wijzig de beschrijving → klik Opslaan
  3. ✓ Verwacht: "✓ Opgeslagen" melding, tabel geüpdatet

Stap 7 — Kaart + Tools (2 min)

Test 7.1: Kaart

  1. Tab Kaart → klik 🎮 Open Mythic Table
  2. ✓ Verwacht: Mythic Table opent in nieuw tabblad

Test 7.2: Dobbelstenen

  1. Tab Tools → klik d20
  2. ✓ Verwacht: resultaat (1-20) verschijnt groot + in geschiedenis
  3. Typ 3d6+2 in het invoerveld → klik 🎲 Gooi!
  4. ✓ Verwacht: resultaat met aparte worpen (bv. [4, 2, 5] = 13)

Test 7.3: THAC0 calculator

  1. THAC0: 17, Vijand AC: 5, Bonussen: 2
  2. ✓ Verwacht: "Je hebt 10+ nodig"

Stap 8 — Wachtwoord reset (2 min)

Test 8.1: Reset voor speler

  1. Klik 🔑 Reset rechtsboven
  2. Kies een speler → stel tijdelijk wachtwoord in (bv. "Test123")
  3. ✓ Verwacht: bevestiging. Speler wordt bij volgende login verplicht nieuw wachtwoord te kiezen.

Stap 9 — Export (2 min)

Test 9.1: Karakter exporteren

  1. Open een karakter → klik 📤 Export
  2. Kies PDF → een nieuw venster opent met printbaar karakterblad
  3. Kies JSON → een .json bestand wordt gedownload
  4. ✓ Verwacht: beide werken zonder foutmeldingen

Alles getest? ✓
