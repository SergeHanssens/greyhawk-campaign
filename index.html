⚔️ Virtual Tabletop — Mythic Table

Jullie DM gebruikt Mythic Table voor het speelveld: kaart met fog-of-war, tokens verplaatsen, en dobbelstenen rollen. Klik om de sessie te openen.

🎮 Open Mythic Table → Speelveld ℹ️ Over Mythic Table
Wat je daar kan: kaart bekijken, je token verplaatsen, fog-of-war (DM toont/verbergt delen), dobbelstenen rollen (meerdere tegelijk), en chat.
Wie beheert: de DM (Bart) — hij tekent de kaarten en regelt de zichtbaarheid.

⚔️ Gevecht & Initiative

Gevechten worden beheerd via de Sessies tab. Open een sessie → klik ⚔️ Start gevecht.

Hoe het werkt:
1. Maak een sessie aan met alle deelnemers (tab Sessies)
2. Open de sessie → klik "⚔️ Start gevecht"
3. Spelers vullen zelf hun initiative-worp in (d6 + DEX mod)
4. DM klikt "🎲 Gooi init voor DM-karakters" (vijanden/NPCs)
5. Sorteer op initiative (laagst eerst = snelst in AD&D)
6. Per ronde: klik "📋 Actie" per karakter om vast te leggen wat ze doen
7. HP aanpassen, status wijzigen (dood/bewusteloos/gevlucht)
8. 🎮 = wie bestuurt het karakter (DM kan overnemen/doorgeven)
🎯 THAC0 Calculator

Voer je THAC0 en de AC van je vijand in. De calculator toont welke D20-worp je minimaal nodig hebt.
